# MCT-175 — Docker Stack Base: compose dev/prod profile + env split + ADR-030 publish + cross-repo lock gate

## §1 동기

**EPIC-mctrader-docker-stack Story-1 (Epic entry)** — compose infrastructure 기반 정비.

`mctrader-hub/compose.yml` 은 인프라 stack (postgres + minio + redis + 7 service) 만 정의. 어플리케이션
(mctrader-data collector + mctrader-engine paper-engine) 은 compose 외부에서 별도 실행 → operational
ambiguity + dev/prod parity 부재.

본 Story 는 Epic 진입 기반 정비:
1. **ADR-030 publish** (docker stack governance singleton — 8 D 본문 박제)
2. **compose profiles dev/prod 분리** (D3): minio/mc service 에 `profiles: ["dev"]`, `.env.dev` / `.env.prod`
   env_file 분기
3. **NAS DNS preflight** (D7): `scripts/preflight-nas-dns.sh` (compose up 전 DNS+TCP+S3 검증)
4. **cross-repo lock CI gate** (D13): `.github/workflows/cross-repo-lock-check.yml` + Python check script
5. **WAL host bind mount 정책 박제** (D1): collector service stub 주석 + runbook stub

## §4 Acceptance Criteria

### AC-1 (D3): compose profile config PASS

```bash
docker compose --profile dev config    # PASS (exit 0, yaml 출력)
docker compose --profile prod config   # PASS (exit 0, yaml 출력)
```

### AC-2 (D3): .env endpoint 분기 정합

- `.env.dev`  : `NAS_MINIO_ENDPOINT=http://minio:9000` (hub MinIO)
- `.env.prod.example` : `NAS_MINIO_ENDPOINT=http://mcnas01.internal.mclayer.it:9000` (NAS)

### AC-3 (D7): preflight script exit code 정합

```bash
./scripts/preflight-nas-dns.sh .env.prod
# exit 0 = DNS+TCP+S3 ALL PASS (NAS endpoint 접근 가능 환경에서)
# exit 10 = DNS resolve FAIL
# exit 20 = TCP connect FAIL
# exit 30 = S3 list FAIL
```

dev 환경 직접 실행 = DNS fail 예상 (정상 — container 내부 hostname `minio` 는 host 측 미해석).

### AC-4 (D1): WAL mount 정책 박제

`compose.yml` 에 collector service stub 주석 포함:
```yaml
# volumes:
#   - /var/lib/mctrader/wal:/var/lib/mctrader/data  # ADR-030 §D1 host bind mount
```

실 활성화 = MCT-176 LAND 시.

### AC-5 (D13): cross-repo lock CI gate 동작

- `.github/workflows/cross-repo-lock-check.yml` hub PR push 시 자동 실행
- `scripts/check_cross_repo_locks.py` exit 0 = 6 repo uv.lock python_version + lib major drift 없음
- exit 1/2 = drift → CI FAIL (merge 차단)
