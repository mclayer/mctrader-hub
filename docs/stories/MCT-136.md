### 변경 내용

**Candle Protocol 재정의** (`mctrader_market/protocols/candle.py`):
- 기존 stored entity 의미 → derived view 의미
- `source` field 추가 (`legacy_candle` / `transaction_derived`)

**Information bar Protocol 신설** (`mctrader_market/protocols/information_bar.py`):
- 신규 — volume/tick/dollar bar 의 abstract Protocol
- 공통 field: `bar_label` (e.g. `vol_1000000krw`), `genesis_ts`, `threshold`, `contract_metadata_version`, `ts_close`, OHLCV

**tick.v1.1 schema 확장** (`mctrader_market/schemas/tick.py`):
- ADR-009 D10 의 8 column 유지
- minor extension 3 column 추가 (ADR-009 §D10.8):
  - `ingest_seq` (uint64) — collector 측 monotonic 발급, dedup + 결정론적 replay key
  - `payload_hash` (string) — raw WS frame SHA256 16-hex
  - `validation_status` (string) — `OK` / `GAP` / `MALFORMED` / `RECONNECT_BOUNDARY`
- 거부 column: `trade_id` (Bithumb 미노출), `maker_or_taker` (검증 안 됨)

## §2 도메인 해석

Candle Protocol 의 의미 변경 — stored entity (Parquet row) → derived view (transaction-derived 또는 legacy). signature 동일 유지 (caller 코드 변경 최소화), `source` field 만 추가하여 provenance 노출. Information bar Protocol = volume/tick/dollar bar 의 공통 추상 — Aggregation Core Lib (Story-3) 의 4 bar 알고리즘 각각의 output 이 본 Protocol 만족.

## §5 요구사항 확장 해석

AC:
1. `Candle` Protocol 에 `source: Literal["legacy_candle", "transaction_derived"]` field 추가 (frozen Protocol)
2. `Candle` Protocol 에 `contract_metadata_version: str` field 추가 (optional, default = "ohlcv.v1" for legacy)
3. `InformationBar` Protocol 신설 — 4 method (volume_bar / tick_bar / dollar_bar / time_bar) 또는 단일 Protocol with `bar_label` discriminator
4. `tick.v1.1` schema = 기존 8 col + 3 신규 col (ingest_seq / payload_hash / validation_status)
5. backward compat — 기존 tick.v1 reader 가 v1.1 row 의 3 신규 column NULL/default 처리 가능
6. tests: Protocol contract + schema lint + backward compat (legacy candle 시뮬레이션)

Edge case:
- legacy candle (Parquet pre-cutoff) load 시 `source = "legacy_candle"` + `contract_metadata_version = "ohlcv.v1"` 자동 인식
- transaction-derived candle 의 contract_metadata_version 은 Aggregation Core Lib (Story-3) 의 contract_id
