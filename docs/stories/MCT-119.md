## 1. 배경

기존 mctrader-engine 은 단일 strategy = 단일 SignalProducer → 단일 Executor 1:1 binding 패턴이었다. 다중 신호 합성·리스크 가드·동적 비중 산출이 strategy 코드 내부에 인라인됐고, 동일 신호 로직이 여러 strategy 에 중복 구현됐다.

본 Story 는 7-layer pipeline 으로 layer 분리:
1. **SignalProducer** — frame 단위 long/short/flat/no_view 시그널 발생
2. **Aggregator** — 다중 시그널 합성 (WeightedSum/And/ThresholdCount)
3. **Constructor** — 자산별 비중 산출 (EqualWeight/FixedWeight/VolatilityParity)
4. **PositionManager** — 현재 포지션 상태 트래킹
5. **RiskConstraint** — pre-trade 리스크 게이트 5종 (FixedSlTp/AtrSl/MaxDD/SizeCap/Composite)
6. **ExecutionPlanner** — OrderIntent 변환
7. **EventStore** — pipeline_events 테이블 7개 이벤트 타입 기록
