### 문제

신규 거래소 추가 시 "engine/market-core/ADR 변경 없이 data 단독으로 활성화 가능"이라는 D5 속성이
코드 상 달성됐으나, 이를 **회귀 방지 test + 운영 절차 문서**로 박제하지 않으면:

1. 향후 개발자가 실수로 engine 에 거래소 의존을 추가해도 CI 가 차단하지 않음
2. 신규 거래소 추가 절차가 문서화되지 않아 새 기여자가 잘못된 레이어에 코드 추가 가능
3. ADR-031 §D5 = Proposed 상태 — VERIFIED 박제 없으면 Epic POLICY_FINALIZED 불완전
