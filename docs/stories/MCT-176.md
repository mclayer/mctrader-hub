## §4 Acceptance Criteria

### AC-1 collector container inspect PASS

```bash
docker compose --profile prod config | grep -A 10 "collector:"
# 기대: collector service 정의 출력 (image, volumes, env_file, depends_on preflight)
docker compose --profile prod up -d collector
docker inspect mctrader-collector | grep '"Status"'
# 기대: "Status": "running"
```

collector service 가 prod profile 하에 기동되고, WAL bind mount
(`/var/lib/mctrader/wal:/var/lib/mctrader/data`) 가 정상 연결됨을 확인.

### AC-2 effective-config subcommand PASS

```bash
docker exec mctrader-collector mctrader-data effective-config --format json
```

출력 JSON 이 아래 3 항목을 포함:

- `nas_endpoint`: `mcnas01.internal.mclayer.it:9000` (prod .env 값)
- `wal_path`: `/var/lib/mctrader/data` (컨테이너 내부 bind mount 경로)
- `universe_id`: env 또는 YAML default 중 우선 적용된 값

source order 출력: `env > YAML default > built-in default` 명시.

### AC-3 rotation dry-run PASS

```bash
bash scripts/rotate-nas-credentials.sh --dry-run
```

exit 0 + stdout 에 아래 항목 출력:

- step 1~5 dry-run 시뮬레이션 로그
- `[DRY-RUN] Slack webhook: would send rotation notice`
- `[DRY-RUN] compose down && compose up -d: skipped`

실제 credential 변경 또는 container restart 없이 절차 확인 가능.

### AC-4 MCT-175 carry over fix verify

P1-2 (preflight DNS wildcard FP) fix 적용 여부 확인:

```bash
bash scripts/preflight-nas-dns.sh
```

exit code 0 (ALL PASS) 또는 exit 10/20/30/99 (정상 fail-fast, 문서 기준 정합).

P2-1 (shell error handling) `rotate-nas-credentials.sh` 반영:

```bash
bash -n scripts/rotate-nas-credentials.sh
# syntax check PASS
head -5 scripts/rotate-nas-credentials.sh | grep -E "set -euo pipefail|set -e"
# set -euo pipefail 포함 확인
```

## §6 Risk 상세

### R-MCT-176-1 secret leak (HIGH — MCT-175 R1 carry over)

- **위협**: `.env.prod` 에 NAS_MINIO_ACCESS_KEY / NAS_MINIO_SECRET_KEY 평문 보관
  + HTTP-only 전송 → 내부망 패킷 스니핑 또는 파일 유출 시 credential 노출
- **mitigation**:
  - `.env.prod` git tracked 금지 (`.gitignore` 박제 MCT-175 LAND 확인)
  - 파일 권한 0600 의무 (`chmod 600 .env.prod`)
  - rotate-nas-credentials.sh 90d 자동화 (본 Story D9 구현)
  - NAS firewall port 9000/9001 IP-allowlist mctrader host only (ADR-027 §D2 mitigation)
- **acceptance**: user_acknowledged_at: 2026-05-15 (MCT-175 §3 박제)

### R-MCT-176-2 rotation script failure (MEDIUM)

- **위협**: `rotate-nas-credentials.sh` 실행 중 MinIO container down + Slack webhook 실패
  → credential 불일치 + 무통보 outage
- **mitigation**:
  - `set -euo pipefail` 적용 + trap ERR rollback 절차 박제
  - Slack webhook 실패 시 GitHub Issue 자동 발의 (`gh issue create`)
  - dry-run 모드 (`--dry-run`) 의무 사전 검증
  - rotation log row append 후 git commit = idempotent audit trail
- **잔여 risk**: outage window (compose down ~ up 완료 구간 ~30s) 발생 가능.
  market close 시간대 실행 권고 (rotation runbook 박제).
