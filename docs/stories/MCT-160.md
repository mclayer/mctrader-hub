### Acceptance Criteria (6 AC + Edge Case 2)

**AC-1 (L2 cadence root fix)**: `L2Compactor.compact_hour` 가 `date_utc` 인자 explicit accept → `_run_l2` caller (runner.py) 가 partition-level latest date discover 후 명시 전달 → KST→UTC date roll edge 시점에도 silent skip 0 (Phase 2 test §8 Test-1 verify)

**AC-2 (L3 cadence root fix, D1 합병)**: `L3Compactor.compact_day` 가 `date_utc` 인자 explicit accept → `_run_l3` caller 명시 전달 → L2 동형 silent-skip 차단 (Phase 2 test §8 Test-2 verify)

**AC-3 (OOM root fix, D3 streaming)**: L2/L3 compactor 의 `pa.concat_tables` 단일 buffer 제거 → chunk-based concat (chunk size = 1024 rows) + `row_group_size=100_000` 명시 `ParquetWriter.write_table` per-chunk loop → 32GB peak + i32 offset 4GB cap overflow 동시 차단 (Phase 2 test §8 Test-3 verify)

**AC-4 (monotonic verify + quarantine, D4)**: L1 산출물 post-write verify — `ts_utc` column monotonic non-decreasing 검증 → 위반 시 산출물을 `quarantine/` directory 로 isolate + Prometheus `compactor_quarantine_total{tier,reason}` emit + 다음 segment 진행 (fail-closed 차단 정책, ADR-009 §D12.2 forward-only invariant 강화)

**AC-5 (R-EXTRA — caller sha256 + data=Path)**: `runner._dispatch_dual_write` 가 caller-side `sha256(parquet_path.read_bytes())` 산출 후 `DualWriter.write(data=parquet_path, sha256=<hex>)` 호출 (data=bytes 전달 폐기, `read_bytes()` 호출 2회 → 1회 + memory 재할당 1회 → 0회). DualWriter 변경 0 (이미 `data: Path | bytes` accept).

**AC-6 (P1 nullability hardening, MCT-162 CodeReviewPL surface 합병)**: l1.py 3 schema (`_TRANSACTION_SCHEMA` + `_ORDERBOOKSNAPSHOT_SCHEMA` + `_ORDERBOOKDEPTH_SCHEMA`) 일관 `pa.field(name, type, nullable=False/True)` 명시. nullable=False 의무 column = `ts_utc` / `exchange` / `symbol` / `side` / `price` / `quantity` (3 schema 공통). nullable=True 허용 = `raw_json` / `node_id` / `collector_run_id` (metadata). ADR-009 amendment trail append. malformed frame 검증 — `if side is None or price is None or quantity is None: raise ValueError(f"malformed {channel} frame at index={i}")` + Prometheus `compactor_malformed_frame_total{channel,exchange}` Counter +1.

**Edge Case 1 (KST→UTC date roll silent skip)**: `_run_l2` 가 `datetime.utcnow()` 호출 시점이 KST 00:00~09:00 사이 → UTC date = 어제 date → 어제 date 의 L1 결과 hit 0 → silent return None (현재 production bug). AC-1 fix 후 caller 가 partition discover → 명시 전달로 차단.

**Edge Case 2 (post-write verify의 quarantine 산출물 재처리 정책)**: quarantine isolate 된 산출물 = 운영자 manual review + 진단 의무 (compactor 자동 retry 0). Story §11 박제 — `quarantine/` directory layout + 재처리 정책 SSOT.

## §2 도메인 해석 (DomainAgent 통합)

### 본 Story 의 domain 위치

mctrader 6-repo 의 **데이터 layer (mctrader-data) — Stage 3 wiring post-deployment 5중 차단 cycle Story-2 (핵심 fix)**. EPIC-compactor-operations 의 Story-1 (MCT-162) LAND 후 sequential 진입. domain 책임:

1. **L2/L3 cadence 정상화** — `_run_l2` / `_run_l3` caller 가 partition-level latest date discover 후 `compact_hour` / `compact_day` 에 명시 전달 (silent skip 차단)
2. **L2/L3 OOM 방어** — `pa.concat_tables` 단일 buffer 제거 → chunk-based concat + `row_group_size` 명시
3. **post-write monotonic invariant** — `ts_utc` non-decreasing verify + quarantine fail-closed
4. **DualWriter memory hot path 개선** — caller sha256 산출 + `data=Path` streaming (memory 재할당 차단)
5. **l1.py 3 schema nullability discipline** — MCT-162 CodeReviewPL P1 finding 합병 (single PR coverage)

### 4 핵심 도메인 사실

1. **L2/L3 cadence resolver SSOT** (CodebaseMapperAgent fetch): `runner.py` `_run_l2` / `_run_l3` 가 cadence 결정 entrypoint. 현재 `hour_utc = datetime.utcnow()` / `date_utc = datetime.utcnow().date()` 하드코딩 → caller 위치에서 date discover 후 `compact_hour(date_utc=...)` / `compact_day(date_utc=...)` explicit 전달 변경.
2. **L1 partition layout SSOT** (ADR-009 §D2.6 + MCT-159 FIX Iter 1 InvariantHarness): `market/{channel}/schema_version={schema}/tier=L1/exchange={ex}/symbol={sym}/date={D}/node={N}/part-{run_id}.parquet`. partition-level latest date lookup = glob `tier=L1/exchange={ex}/symbol={sym}/date=*/` directory listing → max(date) discover.
3. **OOM 패턴 (MCT-156 사례)**: `pa.concat_tables(tables).sort_by("ts_utc").to_pylist()` 의 단일 buffer 32GB peak + raw_json `large_string` 누적 i32 offset 4GB overflow (MCT-162 ADR-009 §D11.9 LargeUtf8 cast 의무 cross-ref). chunk concat + per-chunk write 로 memory 일정 유지.
4. **DualWriter signature 정합** (MCT-151 data#43, MERGED): `DualWriter.write(data: Path | bytes, sha256: str, ...)` 이미 `Path` accept. caller (runner.py) 만 `data=path` + caller sha256 산출 변경.

### Channel SSOT (3 schema nullability discipline)

본 Story 의 ADR-009 amendment trail (D7+D10) = `transaction` / `orderbooksnapshot` / `orderbookdepth` 3 schema 의 nullable field 일관 명시. MCT-159 FIX Iter 1 InvariantHarness channel-aware lookup 자동 적용 — column count 검증과 더불어 nullability invariant 도 박제.
