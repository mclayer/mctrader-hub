### 변경 내용

**Streaming Aggregator** (`mctrader_engine/hot/streaming_aggregator.py`):
- Python asyncio + per-symbol state machine
- bounded queue 격리 (network ingest / WAL append / aggregation / strategy dispatch 분리)
- 50 sym × ~10-200 trades/sec (burst 5000) 처리
- Story-3 Aggregation Core Lib 직접 import

**State Machine** (`mctrader_engine/hot/state_machine.py`):
- per-symbol running OHLC + cumulative threshold accumulator
- KRW scaled integer (원 단위 = naturally integer-friendly)
- ingestion 경계에서만 Decimal → scaled int 변환

### 성능 목표

- p50 < 5 ms bar-close
- p99 < 50 ms bar-close
- per-symbol state 메모리 < 10 KB/sym → 50 sym 전체 < 1 MB

- **BarDispatch async callback contract** — `Callable[[InformationBarModel], Awaitable[None]]`. ADR-008 SemVer MAJOR 전 변경 금지. Story-9 (engine consumer) 가 본 contract 사용.
