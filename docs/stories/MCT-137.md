## §1 사용자 요구사항 (verbatim — 변경 차단)

**범위**: `mctrader-data` repo (`mctrader_data/aggregation/`). Phase 1 (foundation).

### 변경 내용

**Aggregation Core Lib** (`mctrader_data/aggregation/core.py`):
- pure-Python aggregation core — Hot(asyncio) / Cold(DuckDB) 양 path 가 import
- 4 bar 알고리즘 SSOT:
  - **time bar**: 임의 timeframe, boundary `[start, end)` inclusion
  - **volume bar**: cumulative quantity threshold 도달 시 bar close
  - **tick bar**: cumulative trade count threshold 도달 시 bar close
  - **dollar bar**: cumulative price × quantity (KRW notional) threshold 도달 시 bar close
- 수치 표현: KRW scaled integer 또는 명시적 precision Decimal — boundary drift 방지
- per-symbol state: running OHLC + cumulative threshold accumulator + tie-breaking rule

**Contract Metadata** (`mctrader_data/aggregation/contract_metadata.py`):
- immutable metadata structure: `genesis_ts` / `threshold` / `precision` / `rounding_rule` / `source_cutoff` / `tie_breaking` / `version` / `contract_id` (SHA256)
- contract_id 발급 함수
- backward-compat versioning

### Consistency SLO

drift < 0.01% bar count mismatch (Hot vs Cold). 위반 시 strategy 입력 Cold fallback.
