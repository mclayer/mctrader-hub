## §4 주요 기술 결정

- **Signal Worker Docker 제어**: `import docker.errors as _docker_errors` alias — mock contamination 방지 (테스트 중 docker 모듈 패치 시 except 절이 BaseException 상속 안 한 MagicMock을 catch하는 버그 방지)
- **Idempotency-Key**: signal_control에 기존 `control.py`와 동일한 24h SQLite dedupe 패턴 적용
- **Prometheus metrics**: isolated `CollectorRegistry()` — hot reload 시 중복 등록 에러 방지
- **Self-HTTP**: /metrics 엔드포인트가 localhost:7821 /admin/status/* 를 호출해 Gauge 업데이트
- **Polling**: `time.sleep(2) + st.rerun()` (전환 중), `time.sleep(10) + st.rerun()` (안정)
