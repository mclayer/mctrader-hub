- **Status**: 신규

- **대상 Repo**: `mctrader-hub` (스크립트), 6 repo 전체 (정책 적용)

## 1. 배경

MCT-100/101/102 (2026-05-08) + MCT-110 (2026-05-09) 에서 parallel agent branch race가
재발. ADR-019 D1~D5 결정을 구현한다.

## 2. 수행 내용

### 2.1 Preflight bootstrap script

`scripts/agent-preflight.ps1` (Windows PowerShell) 작성:
```powershell
# ADR-019 D2/D3/D5: agent 환경 부트스트랩 표준
# 사용법: . scripts/agent-preflight.ps1 -RepoPath <path> -Branch <branch>

param(
    [string]$RepoPath = ".",
    [string]$Branch = "",
    [switch]$UseWorktree
)

# D2: Python 버전 검증
$pyVersion = py -3.12 --version 2>&1
if ($LASTEXITCODE -ne 0) { throw "py -3.12 not found" }

# D3: Editable install
py -3.12 -m pip install -e $RepoPath --quiet

# D4: Branch 검증
if ($Branch) {
    $current = git -C $RepoPath branch --show-current
    if ($current -ne $Branch) { throw "Branch mismatch: expected $Branch, got $current" }
}
```
