## 1. 배경

Phase 1 (MCT-119) 7-layer pipeline 은 백테스트 시뮬레이션 가능 수준. Phase 2 는 paper/live 모드 운용 시 필수 게이트 5종을 추가:

1. **CostModel** — 수수료·슬리피지 모델 (Backtest vs Live 분리)
2. **ExchangeConstraint** — 거래소별 lot_size·min_notional·tick_size 검증 (BithumbConstraints 우선 구현)
3. **SignalGroup** — 멀티-레그 신호 ALL_OR_NONE 원자성 보장 (PairSignalGroup 헷지 페어 등)
4. **PipelineMonitor** — frame latency·신호 anomaly 감지 (deque 기반)
5. **HotSwapManager** — strategy version 무중단 교체 (prepare/activate/rollback 상태 머신)
