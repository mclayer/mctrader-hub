**AC-4 (MCT-153 손실 박제)**: (a) ADR-027 §D9 축약 amendment LAND (b) MCT-161 reserve cross-link (c) 본 Story §11 데이터 마이그레이션 trace = "local source 보존 → forward-only invariant 위반 0"
