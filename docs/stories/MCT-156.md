**Stage 3 wiring obligation의 본질**: Stage 2 가 production-grade NAS primitive + 운영 layer + historic backfill + read-side cutover 모두 완성했으나, **hot pipeline (compactor) 자체가 NAS endpoint 안 가는 환경에서 운영 중**. 본 Story 가 그 결락을 메우는 entrypoint vertical slice — compose env 전환 + cli.py DualWriter lazy build inject + runner.py L2/L3 DualWriter 호출 wiring + legacy MinioUploader deprecation 마킹.
